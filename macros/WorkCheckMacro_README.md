# WorkCheckMacro（作業・ダブルチェック ワンクリック打刻マクロ）

チェックシートの各行に「作業打刻」「チェック打刻」ボタンを配置し、
ボタンを押すだけで **Windows アカウント名** と **日時** をその行に自動入力する Excel VBA マクロ。
これまで手入力していた氏名・時刻の記入をワンクリック化し、時短と入力ミス防止を実現します。

## シート構成

| 列 | 内容 | 入力方法 |
|---|---|---|
| A列 | 項目名（作業内容） | 手入力 |
| B列 | 作業者 | **ボタンで自動** |
| C列 | 作業日時 | **ボタンで自動** |
| D列 | ダブルチェック者 | **ボタンで自動** |
| E列 | ダブルチェック日時 | **ボタンで自動** |
| F列 | 作業打刻ボタン | 自動配置 |
| G列 | チェック打刻ボタン | 自動配置 |

## ファイル

| ファイル | エンコード | 用途 |
|---|---|---|
| `WorkCheckMacro.bas` | UTF-8 | ソース管理用・macOS 版 Excel でのインポート用 |
| `WorkCheckMacro_Win.bas` | Shift-JIS (CP932) | **Windows 版 Excel** でのインポート用 |

## 文字化けについて
Windows 版 Excel の VBE は `.bas` インポート時にファイルをシステムの
ANSI コードページ（日本語環境では Shift-JIS）として読み込むため、
UTF-8 の `.bas` を読み込むとコメントや文字列が文字化けします。
そのため Windows では必ず **`WorkCheckMacro_Win.bas`（Shift-JIS）** を取り込んでください。

## 導入手順（Windows）
1. Excel で `Alt+F11` → VBE を開く
2. 「ファイル」→「ファイルのインポート」→ `WorkCheckMacro_Win.bas` を選択
3. `SetupSheet` を一度実行（見出しと列幅が整う）
4. A列に項目名（作業内容）を入力する
5. `SetupButtons` を実行 → 各行に「作業打刻」「チェック打刻」ボタンが自動配置される

## 使い方（日常運用）
- 作業が終わった行の **「作業打刻」** ボタンを押す → 作業者(B)・作業日時(C)が入る
- ダブルチェックした行の **「チェック打刻」** ボタンを押す → ダブルチェック者(D)・日時(E)が入る

## マクロ一覧

| マクロ名 | 役割 |
|---|---|
| `SetupSheet` | 見出し・列幅を整える（初回のみ） |
| `SetupButtons` | A列の項目数に合わせて各行にボタンを自動配置（行を増やしたら再実行） |
| `StampWorker` | 作業者・作業日時を打刻（ボタンに割り当て済み） |
| `StampChecker` | ダブルチェック者・チェック日時を打刻（ボタンに割り当て済み） |
| `RemoveButtons` | 配置した打刻ボタンを全削除（作り直したいとき） |

## 仕様メモ
- 氏名は Windows のログインアカウント名を **小文字に揃えて**（例：`m.matsumoto`）入れます。
  大文字混じり（`M.Matsumoto` / `MATSUMOTO`）でも小文字に統一され、表記ゆれを防ぎます。
  表示名（「松本」など）にしたい場合は `GetWindowsUser` を変換方式に差し替えてください。
- 打刻済みのセルでも上書きされます（確認ダイアログは出ないシンプル仕様）。
- 行の特定は `Application.Caller`（押されたボタンの位置）で行うため、ボタン1種につき1マクロで全行に対応します。
- 項目（行）を増やしたら `SetupButtons` を再実行すると新しい行にもボタンが付きます。
