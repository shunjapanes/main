# 文章→HTML変換ツール

テキストをHTMLエンティティに変換するシンプルなWebツールです。

## 機能

- リアルタイムで文章をHTML表記に変換
- 特殊文字（`<`, `>`, `&`, `"`, `'`）を自動的にHTMLエンティティに変換
- 変換結果をクリップボードにコピー
- レスポンシブデザイン（スマホ対応）

## 使い方

1. `index.html` をブラウザで開く
2. 左側の入力欄に文章を入力
3. 右側に自動的にHTML表記が表示されます
4. 「HTMLをコピー」ボタンでクリップボードにコピー可能

## デモ

ブラウザで `index.html` を開くだけで動作します。サーバーは不要です。
